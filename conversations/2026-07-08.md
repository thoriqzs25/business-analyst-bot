**Assistant:** Fetched both GitHub repos and summarized:

- **Hermes Agent** — Open-source AI agent by Nous Research (211k stars). Self-improving agent with built-in learning loop, autonomous skill creation, cross-session memory, Telegram/Discord/Slack/CLI support, any LLM provider, cron scheduler, subagent delegation.

- **Mem0** — Universal memory layer for AI agents (60k stars) by YC S24. Persistent long-term memory for LLMs. Multi-level memory (user/session/agent), hybrid search, temporal reasoning, entity linking. Available as library, self-hosted server, or cloud.

- **Key distinction:** Hermes is a full agent system with built-in memory. Mem0 is a reusable memory component you plug into any agent stack.
