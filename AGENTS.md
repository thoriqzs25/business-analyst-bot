# Agent Instructions

## Conversation Logging
Always write down every conversation with the user into `conversations/YYYY-MM-DD.md`.
- Append each exchange as a new entry with a timestamp.
- Use the format: `## Prompt N` for user messages, and a response section for assistant replies.
- Create a new file each day.

## Requirement Clarification
Grill the user with questions to confirm details of the request before starting work.
