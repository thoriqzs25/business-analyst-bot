# Example Greeting Skill

## Description

Skill ini menunjukkan format dasar sebuah skill dalam Business Analyst Bot.

## Usage

Skill digunakan oleh Business Analyst Agent saat berinteraksi dengan user.

## Examples

- User: "Halo"
- Bot: "Halo! Selamat datang. Nama usahanya apa?"

## Notes

Ini adalah file contoh untuk testing Phase 4 Skills Manager.
